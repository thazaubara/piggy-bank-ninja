# piggy-bank-ninja

✔︎ Upload Workflow:

1. download new CSV from banking
2. put in corresponding folder. Rename.
3. Start program and do "ul" to upload to server.

✘ Tagging Workflow:

1. auto tag with tag-auto
2. list all without tags
3. add tags for those
4. auto tag again
5. repeat
